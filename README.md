## 🌟 Overview

**Pegasus Recovery** adalah alat pemulihan sistem dan data yang dirancang untuk kecepatan, keandalan, dan kemudahan penggunaan. Terinspirasi dari kuda terbang mitologi Yunani, Pegasus Recovery siap membawa data Anda kembali dari jurang kehilangan.

---

## ✨ Features

| Fitur | Keterangan |
|-------|-------------|
| ⚡ **Superfast Scan** | Pemindaian sektor disk hingga 10x lebih cepat |
| 🧠 **Smart Recovery** | Deteksi cerdas file yang dapat dipulihkan |
| 🔒 **Secure Mode** | Lindungi data yang sudah dipulihkan dari overwrite |
| 📁 **Multi-Format** | Support 100+ format file (doc, jpg, mp4, zip, dll) |
| 💻 **Cross-Platform** | Windows, Linux, macOS |
| 🧪 **Preview Before Recovery** | Lihat file sebelum dipulihkan |

## 🚀 Quick Start

### Prasyarat
- Python 3.8+ atau Node.js 16+
- 2GB RAM minimum
- Sistem operasi modern

### Instalasi

```bash
# Clone repository
git clone https://github.com/yourusername/pegasus-recovery.git
cd pegasus-recovery

# Install dependencies
npm install
# atau
pip install -r requirements.txt

# Jalankan
npm start
# atau
python main.py
